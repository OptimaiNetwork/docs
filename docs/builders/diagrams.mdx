## Search Request Flow

```mermaid
sequenceDiagram
  participant User
  participant Search as OptimAI Search
  participant Orchestrator
  participant Nodes as OptimAI Nodes
  participant RDL as Reinforcement Data Layer
  participant Agent as Answer Agent

  User->>Search: Submit query
  Search->>Orchestrator: Create retrieval task
  Orchestrator->>Nodes: Fetch or refresh sources
  Nodes-->>Orchestrator: Return candidate sources
  Orchestrator->>RDL: Score freshness, quality, and provenance
  RDL-->>Agent: Ranked context
  Agent-->>Search: Synthesized answer with citations
  Search-->>User: Answer, sources, images, follow-ups
```
